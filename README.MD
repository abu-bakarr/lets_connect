# Installation
-- To Start Backend (Nodejs Server) --
`git clone https://github.com/abu-bakarr/lets_connect.git`
`cd Backend`
`npm install`
`npm start`

-- To Start Frontend (ReactJS) --
`git clone https://github.com/abu-bakarr/lets_connect.git`
`cd Backend`
`cd client`
`npm install`
`npm start`
